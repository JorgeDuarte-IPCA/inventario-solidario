Controlo de frequência de pedidos

Intervalo mínimo (em dias) entre pedidos do mesmo beneficiário. Use 0 para desativar.

Envio de email via Brevo (recomendado online)

O alojamento gratuito (Render) bloqueia o envio por SMTP. Use o Brevo, que envia por API e funciona online. Crie conta grátis em brevo.com, verifique o seu email como remetente, gere uma chave de API (SMTP & API → API Keys) e cole aqui.

Servidor de email (SMTP)

Alternativa ao Brevo (funciona em alojamento que permita SMTP, ou localmente). Configure o servidor de envio de emails. Exemplo para Gmail: servidor smtp.gmail.com, porta 587, e uma palavra-passe de aplicação (não a password normal da conta).

Testar envio

Envia um email de teste para confirmar que a configuração funciona.
